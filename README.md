# xD-keyboard
xD keyboard is nRF52 based wireless mechanical keyboard equipped with Gateron KS-33 low profile key switches. 

# PCB 

Generated with KLEPCBGEN. Designed with both Kicad & Easyeda PRO.

# Case & Plate 

Generated with Swillkb & Keebio. Designed with Fusion 360. Chopped with Bamboo lab Studio.

# Bootloader & Firmware

Bootloader generated from Adafruit_nRF52_Bootloader & Firmware made with ZMK with Custom Board.
